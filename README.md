# 🔐 Crypto & Integrity Cipher Suite

A Python-based cybersecurity toolkit that demonstrates core cryptography and data integrity concepts. This project includes classical encryption techniques, cryptographic hashing, and hash comparison for integrity verification.

---

## 📌 Features

- 🔒 Caesar Cipher Encryption
- 🔓 Caesar Cipher Decryption
- 🔑 Vigenère Cipher Encryption
- 🔐 SHA-256 Hash Generator
- 📝 MD5 Hash Generator
- ✅ Hash Comparison for Integrity Verification

---

## 🛠 Technologies Used

- Python
- hashlib

---

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Clone this repository.
3. Run:

```bash
python crypto_suite.py
```

## 📖 Example

```
===================================
 Crypto & Integrity Cipher Suite

1. Caesar Encrypt
2. Caesar Decrypt
3. Vigenère Encrypt
4. Generate SHA-256
5. Generate MD5
6. Compare Hashes
7. Exit
```
