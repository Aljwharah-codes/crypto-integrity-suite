# Crypto & Integrity Cipher Suite 🔐

## Description
A Python toolkit for encryption, hashing, and data integrity verification.

## Features
- Caesar Cipher Encryption
- Caesar Cipher Decryption
- SHA-256 Hash Generator
- MD5 Hash Generator
- Hash Comparison

## Technologies Used
- Python
- hashlib
